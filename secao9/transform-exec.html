Entendendo a propreidade Transform em CSS

Utilizando essa properiedade é possivel manipular um elemento (dimensionar, mover,rotacionar,etc.)

Rotate

É utilizado para rotacionar de 0 a 360 graus. Utilizando valores positivos, a rotação será feita noo sentido horario, e com valores negativos será feito no sentido anti-horario.

Rotate

Scale

Utilizada para redimensionar, tendo como escala padrão 1. Valores maiores que 1 aumentarão o tamanho e vice-versa.

Scale

Translate

Essa função serve para mover o elemnto no eixos X e Y, ou seja para os lados e para cima e para baixo. A sintaxe pode ser especifica para um dos eixos X/Y (translateX ou translateY), como pode ser apenas translate(x,y).

Translate

Skew

É utlizado para "distorcer" os elementos, alterando os ângulos.

Skew
