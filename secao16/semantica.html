Um pouco mais de semântica

Relembrar

Algumas tags semânticas já vistas

  • cite: fazer algumas referencia na página
  • blockquote: fazer uma citação
  • addres: contato do dono do artigo ou do site
  • code: permite mostrar códigos na nossa página
  • figure e figcaption: criar um conteudo que possa sser removido do conteudo principal sem prejudicar o restante

Novas tag semânticas

Novas tags que permitem tornar nossa página mais semântica

  • abbr: permite mostrar o texto em abreviações ou siglas, isso qaundo passamos o nosso mouse por cima
  • time: mostra para o browser que o foi digitado é uma data
  • del: permite mostrar que uma palavra, frase ou texto foi removida do texto, mas ao invés de tira-la, é feito um risco no que quer ser "deletado", isso permite que motores de busca possam ver tal palavra
  • ins: permite mostrar qual a palavra,frase ou texto foi inserida no lugar daquela que recebeu a tag del
  • kbd: permite mostrar uma formatação diferente para teclas
